# Code of Conduct

Maximus is an open source project, and we want it to feel welcoming, useful, and respectful for contributors of all experience levels.

## Our Standard

When participating in this project, please:

- Be respectful and constructive.
- Assume good intent, but listen when impact is pointed out.
- Keep technical disagreement focused on the code, design, and evidence.
- Help make the project approachable for new contributors.

## Unacceptable Behavior

The following behavior is not welcome here:

- Harassment, threats, intimidation, or stalking
- Hate speech or discriminatory language
- Personal attacks, insults, or repeated hostile behavior
- Publishing private information without permission
- Spam, trolling, or deliberate disruption of discussions

## Scope

This code of conduct applies to repository discussions, issues, pull requests, reviews, and other project spaces managed by the maintainers.

## Enforcement

Project maintainers may remove comments, close discussions, reject contributions, or block participants whose behavior violates this code of conduct.

## Reporting

If you experience or witness unacceptable behavior, please report it privately by email: [soundbrokaz@kakao.com](mailto:soundbrokaz@kakao.com).

This is currently a single-maintainer project. If your concern involves the maintainer, or you are not comfortable using the project email, please use GitHub's platform-level abuse reporting flow instead: [Reporting abuse or spam](https://docs.github.com/en/communities/maintaining-your-safety-on-github/reporting-abuse-or-spam).

Reports will be reviewed as promptly and fairly as possible.
